==================================================================
   TROUBLESHOOTING
==================================================================

If build fails:
- Check if render-build.sh has execute permissions
- View logs in Render dashboard for specific error
- Make sure all files were pushed to GitHub

If 500 error:
- Verify APP_KEY is generated in environment variables
- Check logs for detailed error message

If database error:
- Ensure PostgreSQL database is created and running
- Verify DB_* variables are linked to database
